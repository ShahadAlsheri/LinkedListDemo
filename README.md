# Linked List 

## Description
This project demonstrates the use of a **doubly linked list** in Java with the following operations:
- Insert nodes
- Delete nodes
- Search nodes
- Print all nodes
- Sum of all nodes
- Modify all node values
- Swap nodes
- Sort the linked list
- Additional operations: delete node after, print even nodes, find second-to-last node, find largest node, count total nodes
